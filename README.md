# pytest-hooks-talk
My talk about pytest hooks

Slides here: https://speakerdeck.com/rafaelhenrique/pra-que-servem-o-diacho-dos-hooks-do-pytest

# Install

Install requirements:

```
python3 -m venv .venv
source ~/.venv/bin/activate
pip install -r requirements.txt
```

Run pytest:

```
pytest .
```
